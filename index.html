Reponsive Grid :

In this example, we can see how the items pass to the next row when the size of the column is less than 150px.
We only apply the the property "auto-fit, minmax(150px, 1fr)" in "grid-template-columns.

Results:

Item 1
Item 2
Item 3
Item 4

Code:

.content-box{
display: grid;
grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
gap: 10px;
}
